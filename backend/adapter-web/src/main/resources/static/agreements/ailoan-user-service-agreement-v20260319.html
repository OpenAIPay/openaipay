爱借用户服务协议

版本：2026-03-19

欢迎您使用爱付爱借服务。请在开通前认真阅读并理解本协议的全部条款。

一、服务说明

1. 爱借服务为用户提供现金借款额度及借款、还款能力。

2. 实际可借额度、利率、期限与还款计划，以页面展示和借款确认信息为准。

二、借款与还款规则

1. 您应在约定还款日按时足额还款，逾期将可能产生相应费用并影响信用记录。

2. 平台可基于风险控制、监管要求和履约表现调整授信策略。

三、授权与合规

1. 您同意平台在授信评估、反欺诈、贷后管理中依法处理必要信息。

2. 平台将依据相关法律法规开展服务并履行告知义务。

四、协议生效

您点击“同意并开通爱借”即视为已阅读并接受本协议，自确认时生效。

说明：该页面用于爱借开通签约展示，后续如有法务长文更新，以最新模板版本为准。
