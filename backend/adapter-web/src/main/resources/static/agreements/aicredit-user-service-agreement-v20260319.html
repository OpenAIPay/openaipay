爱花用户服务协议

版本：2026-03-19

欢迎您使用爱付爱花服务。为保障您的合法权益，请在开通前认真阅读本协议。

一、服务内容

1. 爱花服务为用户提供消费信用额度，在爱付支持的交易场景中可先消费后还款。

2. 具体可用额度、账单日及还款日以系统实际展示为准。

二、使用规则

1. 您应在还款日或之前完成还款，逾期可能产生相应费用并影响信用评估。

2. 平台有权依据风险管理规则调整额度、冻结或关闭服务。

三、信息授权

1. 您同意平台在开通及使用期间，基于风控、授信和合规目的处理必要信息。

2. 相关信息处理遵循平台隐私政策和适用法律法规。

四、协议生效

您点击“同意并开通爱花”即视为已阅读并同意本协议，自确认时生效。
